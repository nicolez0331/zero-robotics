Two Kinds of Blocks

Knowing the difference makes snapping intuitive.

C
Command blocks (green, blue, teal)
Stack vertically inside a C-block body — they run top to bottom.
R
Reporter blocks (orange, purple)
Fill oval input slots inside command blocks — they return a value.
✕
Invalid snaps are blocked
Reporters can't go into body areas. Commands can't fill value slots.
